- `replit-prototype-question-bank-mobile-20260629.png`：迁移 Replit 原型后的题库中心移动宽度截图，验证移动顶部导航、筛选、新建导入、新建题目和题目卡片纵向布局。

- `replit-prototype-papers-desktop-20260629-latest.png`：拉取原型 `70fb228` 后的组卷中心桌面截图，验证试卷名称搜索、学科/年级筛选和试卷卡片布局。

当前端界面发生变化时，需要在本目录添加或替换截图，并同步更新本说明。二期界面采用后台管理布局，左侧导航固定包含题库中心、组卷中心、知识点库。

左侧导航支持收缩为图标栏。宽屏下用户可手动收起或展开；展开态显示独立收起按钮，收起态隐藏独立按钮，顶部 logo 作为展开入口。中等窗口下默认进入真实收起状态并使用窄图标栏，避免导航栏挤压导入 OCR 工作台和人工校验编辑器。

当前视觉样式已按 Replit 原型和外部 UI 设计说明升级为“深邃蓝 + 暖橙点缀 + 玻璃拟态侧栏 + 现代卡片”的后台风格。页面功能、接口、导航层级和按钮行为保持不变，前端默认通过 Java backend `8018` 访问现有业务接口。

导入 OCR 工作台必须保证左侧源文件预览区有明确高度，PDF 使用内嵌预览，图片按容器宽度自适应展示。右侧题目校验编辑状态采用稳定纵向布局，避免在窄工作台中把 Markdown 源码、实时预览和元数据表单挤压重叠。

宽屏下题库业务页不再限制在中间窄容器内，而是使用全宽工作台。导入 OCR 工作台会根据可用容器宽度自动决定原文件预览区和人工校验区左右或上下布局；打开人工校验时，Markdown + LaTeX 源码和渲染预览也根据编辑器容器宽度自动左右或上下排布，下方题型、难度、分值、知识点、答案和解析等配置项自动换列。

所有页面必须随浏览器窗口和组件容器自动适配。导入 OCR 工作台在空间不足时从多列自动降级为上下布局；Markdown 源码、公式预览、题目配置表单和操作按钮需要在各自容器内换行或滚动，不能互相遮挡，也不能把右侧内容挤出窗口。

题库中心题目列表顶部使用截图样式筛选面板。第一行依次展示题型、难度、知识点、学科；第二行依次展示年级、地区、年份和“清空筛选”。题型、难度、知识点为下拉框，学科、年级、地区、年份为输入框。

题库中心页面行高必须按内容排列，不能把浏览器视口剩余高度分摊到标题、页签和筛选面板之间；从组卷中心、知识点库或导入 OCR 工作台切回题库列表后，筛选面板仍应紧跟页签显示。

题库中心内的“导入 OCR 工作台”合并导入任务和 OCR 校验：创建导入任务后直接进入工作台，左侧预览试卷或答案原文件，右侧完成题目校验和入库。OCR 结果区要求展示渲染后的数学公式，并把 Markdown 中的题图插入到对应位置。

人工校验卡片中的答案和解析来自 AI 元数据补全时，应展示答案解析匹配证据或复核提示。若同卷 OCR 文本中没有对应题的答案/解析证据，该题仍显示为待补充，不能把模型推理结果当作教师原答案。

导入 OCR 工作台左侧任务列表支持多选批量删除。任务卡片左侧显示复选框，列表顶部显示全选、已选数量和批量删除按钮；批量删除按钮在未选择任务时禁用，不能挤压任务标题、状态和单项编辑按钮。

当前右侧主视图已调整为题目结构化视图：按大题分组，每道题独立卡片展示题干、题图、选项、子题和题型扩展内容，并在题目区顶部显示拆题来源和公式校验汇总。

题图必须随题干一起渲染。导入校验、题库列表和组卷选题池都使用题目的 `images` 字段生成 Markdown 图片资源，保存或入库时同步保留图片数组和题干 Markdown 图片引用。

组卷中心按照设计图拆成试卷列表、新建试卷选题和试卷编辑器三种界面。试卷列表顶部提供学科、年级筛选，卡片展示学科、年级、状态、题目数、满分和导出操作；选题页包含搜索、筛选、已选数量、清空已选和分页题目池；试卷编辑器上方维护试卷标题、学科、年级、试卷头和预览，下方维护题目顺序、分值、移除和追加题目。二次编辑试卷时，追加题目区域必须复用新建试卷的完整题目池交互；发布按钮必须先打开发布前预览弹层，确认后才发布。

最新原型中组卷中心试卷列表顶部新增“按试卷名称搜索”，可与学科和年级联合筛选；搜索框、学科输入、年级输入必须保持同一行稳定对齐，窄窗口下允许换行但不能遮挡按钮。

知识点库按照设计图使用单页列表加弹窗维护。顶部提供搜索筛选和“新建知识点”，列表展示名称、学科、年级和说明，新增和编辑都在居中弹窗中完成。

题目卡片支持 Markdown + LaTeX 双栏编辑：左侧源码，右侧预览，并提供 AI 标准化、AI 解析和保存操作。AI 标准化结果必须先作为候选源码和候选预览展示，用户手动应用后才覆盖编辑内容；如果后端检测到严重公式损坏或已参考原始 OCR，编辑区必须显示对应复核提示。选择题可通过 LaTeX `tasks` 环境直接表达选项。
